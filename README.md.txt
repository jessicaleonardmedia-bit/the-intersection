# The Intersection - Community TV Show Website

A simple, elegant website for "The Intersection" community television show.

## Features
- Responsive design
- Easy-to-use CMS admin panel
- YouTube video embedding
- Social media integration
- Contact form

## How to Update Content

Once deployed, visit: `your-website-url.com/admin`

Login with your credentials and you'll be able to update:
- Homepage content
- Episodes
- About page
- Information page

## Local Development

Open `index.html` in your browser to preview the site.
